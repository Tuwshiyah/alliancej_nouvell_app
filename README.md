# Alliance Journaliere PWA

Cette version web est separee de l'application mobile Expo.

Lancer en local :

```sh
node web-pwa/server.mjs
```

URL locale :

```txt
http://localhost:4173
```

La PWA utilise uniquement la cle publique Supabase et appelle les Edge Functions deja deployees pour creer et verifier les paiements.
